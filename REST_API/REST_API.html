Get Archive List

HTTP request GET to http://[mimimimioon]/Archive/?room=[room]&link=[link]

    Query List

  • room => to search for particular room name
  • link => to search particular stream link (just use YT_[video ID] the YT_ will be automatically parsed to "https://www.youtube.com/watch?v=")

Test command : curl -X GET http://[mimimimioon]/Archive/?room=MonMon%20TL
